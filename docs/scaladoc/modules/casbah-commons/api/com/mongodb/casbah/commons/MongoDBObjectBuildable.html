MongoDBObjectBuildable

trait MongoDBObjectBuildable extends DBObject

DBObject implementing trait meant to proxy as a builder for deserialization of DBObjects. This should cut out some of the 'middle man' in reading MongoDB Objects as the object takes the callback setters directly.
